# Mobile Automation (APPIUM)


##GENERAL INFO

This project shows a simple use of a test automation tool to automate tests for a mobile application. In this project I used the "android-ump-mobile-debug-67a35ff-1.apk" file to try and demonstrate basic navigation and basic functionality.

##TOOLS

* Appium
* Java
* Maven
* TestNG
* Eclipse

##SETUP
* Installed JDK, Android Studio and Android SDKs and set the environment variables
* Installed Eclipse, Maven, TestNG, Appium, Appium Doctor and created a Maven Project
* Added the dependecies in pom.xml file, did clean and build to get these from the Maven repo
* The apk file in "...GlobalKineticAssessment2/src/test/resources/apps/android-ump-mobile-debug-67a35ff-1.apk"
